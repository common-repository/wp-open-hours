WP Open Hours give your customers definite answer when you will be open or closed!

== Description ==

You can use WP Opening Hours in any way that you would like! It comes with four easy-to-use widgets, one for displaying the opening hours, one for special dates/holidays, one for a manual I am Available / Open (or not) image and another for display an image for we are open or we are closed.

12 bright designs and advanced features this widget is a great add-on for any wordpress website.

Visit Demo Link [http://topfreelancers.esy.es/wp-open-hour-widget/](http://topfreelancers.esy.es/wp-open-hour-widget/)

== Installation ==
1. Ensure you have compatible version of Wordpress installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add open text,close text and time table details via Setting->WP Hour
5. Add widget to your sidebar with selecting awsome pre designs.
